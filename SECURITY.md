# Usalama — Security Policy

**SNIL — Swahili Native Intent Language** · Laetoli Ltd

## Kuhusu — Scope

SNIL ni lugha ya programu ya kielimu na *playground*. Mkalimani huendesha katika
kivinjari, na CLI huendesha nje ya mtandao (*offline*) kwenye kompyuta yako. SNIL
haitumii akaunti, haikusanyi data ya watumiaji, na haihitaji muunganisho wa
mtandao ili kufanya kazi. (*SNIL is an educational programming language and
playground. The interpreter runs in the browser; the CLI runs offline on your own
machine. It has no accounts, collects no user data, and needs no network to run.*)

Hata hivyo, kumbuka:

- **Codegen ya Python na CLI** huandika faili na huendesha programu kwenye mashine
  yako. Endesha tu programu za SNIL unazoziamini, kama ungefanya kwa msimbo wowote.
- Kiendelezi cha VS Code (`vscode-snil/`) hutoa *syntax highlighting* tu;
  hakitekelezi msimbo.

## Kuripoti tatizo — Reporting a vulnerability

Ukiona tatizo la usalama (mfano, njia ya msimbo wa SNIL kusababisha tabia
isiyotarajiwa kwenye mwenyeji):

1. **Usifungue *issue* ya hadhara** ikiwa tatizo ni nyeti.
2. Wasiliana na timu ya Laetoli kwa **noreply@laetoli.africa** ukiwa na:
   - maelezo ya tatizo,
   - programu ndogo ya SNIL inayolionyesha,
   - matokeo unayotarajia dhidi ya yanayotokea.

Tutathibitisha mapokezi na kushughulikia ripoti kwa busara. Asante kwa kusaidia
kuweka SNIL salama kwa wanafunzi na walimu.
